إيه هي لغة HTML؟

HTML هي اللغة اللي بنستخدمها علشان نبني صفحات الإنترنت تخيلي الإنترنت زي كتاب، وHTML هي اللغة اللي بنكتب بيها صفحات الكتاب ده.

هل HTML لغة برمجة؟

لأ، HTML مش لغة برمجة، هي لغة وصف. يعني بتقول للمتصفح: "حط عنوان هنا، صورة هنا، فقرة هنا..." يعني بترتّب وتنظّم المحتوى، مش بتعمل حسابات أو عمليات برمجية.

HTML بتشتغل إزاي؟

HTML بتشتغل باستخدام حاجة اسمها الوسوم (Tags)

ودي أوامر صغيرة بتقول للمتصفح يضيف حاجات معينة زي:

عنوان كبير

فقرة نص

صورة

جدول

رابط

كل وسم بيبقى بين قوسين زي كده: < >

إيه هي مكونات صفحة HTML؟

صفحة HTML بتتكوّن من 3 أجزاء رئيسية:

الرأس (head): فيه معلومات مش بتظهر، بس مهمة للمتصفح.

العنوان (title): اسم الصفحة اللي بيظهر فوق في المتصفح.

الجسم (body): ده الجزء اللي بنشوفه فعلاً على الصفحة: صور، نصوص، روابط... إلخ.

ليه نستخدم HTML؟

علشان HTML بتخلّينا:

نكتب نصوص في صفحة الإنترنت

نضيف صور وفيديوهات

نعمل روابط تودّي لصفحات أو مواقع تانية

ننظّم الصفحة باستخدام جداول، قوائم، عناوين

نعمل هيكل مرتب وواضح يساعد الزائر يفهم المحتوى بسهولة

إزاي HTML بتكون موجودة في أي موقع؟

لما بتفتحي أي موقع، المتصفح بيكون بيقرأ ملف HTML عشان يعرف إيه يعرض،

مثلاً:

يكتب عنوان

يعرض صورة

يفتح رابط

كل ده عن طريق أكواد HTML.
