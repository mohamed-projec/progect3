إيه هي CSS؟

CSS هي اختصار لـ Cascading Style Sheets، وده يعني "صفحات الأنماط المتسلسلة".

ودي اللغة اللي بنستخدمها علشان نزيّن ونجمّل شكل صفحات الإنترنت.

يعني إيه CSS؟

تخيلي إن HTML هي جسم الإنسان، وCSS هي اللبس والمكياج والتنسيق اللي بيخليه شكله شيك.

CSS بتضيف الألوان، الترتيب، الحركات، وكل حاجة تخلي الصفحة شكلها جذاب.

تضيف أنيميشن وحركات جميلة

إيه الفرق بين HTML و CSS؟

HTML = المحتوى (نصوص، صور، روابط)

CSS = الشكل (ألوان، خطوط، ترتيب، حركات)

مثال بسيط:

HTML زي الحائط الأبيض، وCSS هي اللي بتلوّن الحائط وتحط عليه لوحات وستاير.
